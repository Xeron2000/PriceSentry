# PriceSentry Documentation

Welcome to PriceSentry documentation! This directory contains detailed guides for configuring and using PriceSentry.

## Available Languages

All documentation is available in both English and Chinese:

| Document | English | 中文 |

## Documentation Structure

- **Configuration Guide** - Detailed instructions for configuring PriceSentry, including exchange setup, notification channels, and advanced features
- **Main README** - Project overview, quick start guide, and common commands

## Default Language

English is the default language for all documentation. You can switch between languages using the links at the top of each document.
